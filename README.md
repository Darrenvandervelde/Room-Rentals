# Room-Rentals
UI / UX for Room Rentals Page for Booking. 

![ScreenShot](Screenshot.png)

[View live Demo](https://room-rentals-nu.vercel.app/)

## Overview

RoomRentals is a modern, responsive room rental and booking website designed to showcase available rooms and provide users with a smooth browsing experience. The project focuses on clean UI design, mobile responsiveness, and scalability for future booking and backend integrations.

This project is ideal as a frontend foundation for a real-world accommodation, rental, or property booking platform.

## Technology

- HTML5 – Semantic structure
- CSS3 – Custom styling & animations
